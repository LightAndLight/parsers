parsers
=======

[![Hackage](https://img.shields.io/hackage/v/parsers.svg)](https://hackage.haskell.org/package/parsers) [![Build Status](https://github.com/lightandlight/parsers/workflows/Haskell-CI/badge.svg)](https://github.com/lightandlight/parsers/actions?query=workflow%3AHaskell-CI)

Goals
-----

This library provides convenient combinators for working with and building parsing combinator libraries.

Given a few simple instances, you get access to a large number of canned definitions.
